# Custom Faces

To use your own photos for the players:

1.  Place 6 PNG images in this folder.
2.  Name them:
    - `face1.png`
    - `face2.png`
    - `face3.png`
    - `face4.png`
    - `face5.png`
    - `face6.png`
3.  Ideally, these images should be transparent PNGs of just the head, about 200x200 pixels.
